# Tribble
GO - Efficient CPU Stat via cmd

## Description
A CPU stat at your command line!

## Requirements
1. Go 1.4.2 (Tested)
1. Gb build (latest)

## Install

To install, use `gb`:

```bash
$ git clone
$ gb build
```
## Usage
After installation

```bash
$ cd bin/
$ ./tribble
```
